[tomixy's biography](/projects/tomixyz-biography)というポートフォリオサイトを長く運用してきましたが、Astroのテーマを上書きしていたこともあり、だんだんとスタイルや機能の拡張が難しくなってきていました。

そんな頃、サイトで利用していた技術にも大きな変化が訪れました。
コンテンツコレクション周りが大きく進化した[Astro 5.0](https://astro.build/blog/astro-5/)、シンタックスが[Rune](https://svelte.jp/docs/svelte/what-are-runes)によって一新された[Svelte 5](https://svelte.jp/)、設定ファイル周りの仕様も大きく変わった[Tailwind CSS v4.0](https://tailwindcss.com/blog/tailwindcss-v4)、…

これらが出揃ったタイミングで、tomixy's biographyを最新バージョンの技術で一から作り直すことを決めました。
そうしてできたのが、当サイトです。

Astroをベースとするのは変わらず、コンテンツコレクションを最大限に活用するようにデータを整理しました。

## Tailwind CSSを剥がす

新しいCSSの機能による設計を積極的に試す場にもしたいため、Tailwind CSSは剥がし、スタイルはすべて素のCSSでリライトしました。
layerやlight-darkなどのモダンなCSSにより、v1と比べてスタイル管理の見通しが良くなった。
